## 1. Recursos necessários para Projeto:

 - Acesso ao Google Colab
 - Uso das linguagens Python e JavaScript
 
---

## 2. Projeto dividido em 4 etapas:

### 2.1. Gravação de áudio do usuário
 - Implementação do gravador de áudio
 - Uso de JavaScrip e Python

### 2.2. Reconhecimento de fala com Whisper
 - Implementação do reconhecimento de fala
 - Transcrição do áudio em texto
 - Speech-To-Text da OpenAI

### 2.3. Integração com API do ChatGPT
 - Integração coma a API da OpenAI
 - Uso de modelos gpt

### 2.4. Sintetizando voz usando gTTS
 - Implementação de sintetização de voz
 - Conversão da resposta para fala 
 - Text-To-Speech da Google
