## Objetivo Bootcamp

Aplicar conhecimentos para analisar e organizar dados, usando inteligência artificial generativa, SQL, Python, Excel, Power Query e Microsoft Copilot.

---

## Tecnologias Utilizadas

- Python
- SQL
- Excel
- Power Query
- Microsoft Copilot
- MySql (alternativa ao MariaDB)
- NotebookLM
- ChatGPT / Whisper
